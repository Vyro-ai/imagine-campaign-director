# Shot Grammar

Shot grammar is the difference between a prompt list and direction.

## Shot Fields

For each shot, define:

- purpose
- duration
- source type
- shot size
- lens feel
- angle
- camera movement
- subject action
- material or product detail
- transition logic
- artifact risk

## Useful Shot Types

### Iconic Opener

One strong image that establishes the world immediately. Use clean silhouette, bold posture, unusual environment, or product geometry.

### Hero Reveal

Introduce the product or subject with legible shape and clean hierarchy.

### Macro Proof

Show texture, craft, label-free product detail, surface behavior, or sensory evidence.

### Environmental Breath

Give the edit space: architecture, weather, moving light, fabric, water, machinery, or empty set detail.

### Performance Beat

Use posture, turn, walk, glance, hand placement, or fabric movement to make the image alive.

### Final Hero Hold

End with a clean frame that can carry product, brand mark, tagline, or CTA in post.

## Rule

If two shots have the same size, same movement, same subject, and same visual job, one of them is probably unnecessary.
