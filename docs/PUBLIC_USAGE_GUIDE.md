# Public Usage Guide

1. Copy or clone this repo.
2. Add your own brand system in `docs/BRAND_SYSTEM_PLACEHOLDER.md`.
3. Provide any starting input: brief, prompt, product photo, style image, mood board, existing video, or mixed assets.
4. Ask your agent to create a comprehensive Imagine.Art Workflow plan.
5. Upload/replace assets in the workflow source section.
6. Generate stillframe anchors.
7. Approve stillframes before motion.
8. Generate motion clips from approved anchors.
9. Generate music using the Imagine.Art Music Studio prompt.
10. Assemble timing, captions, product closeout, and export notes.
11. Run QC.
12. Revise.
13. Package delivery notes.
