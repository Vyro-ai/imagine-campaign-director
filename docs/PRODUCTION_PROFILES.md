# Production Profiles

Production profiles are not creative templates. They do not choose the film, tone, plot, world, shot ladder, or edit structure.

Use profiles only to identify what production must protect:

- product, garment, face, label, UI, or material truth
- continuity locks required before dependent generation
- artifact risks
- category cliches to refuse
- compatible edit forms from `config/editing_structures.json`
- direction DNA suggestions from `docs/REFERENCE_ANALYSIS_METHOD.md`

The creative idea must come from `docs/CREATIVE_DIRECTION_GATE.md`. The production profile only keeps that idea executable.

## How To Use

1. Select the closest profile from `config/production_profiles.json`.
2. Copy its `protect`, `continuity_locks`, `artifact_risks`, and `forbidden_category_cliches` into the treatment.
3. Choose the edit form from the forced-collision premise, not from the profile.
4. If a profile cliche appears in the selected premise, document why it is earned or replace it.

## Current Profiles

If a request spans multiple categories, use one primary profile and pull only specific risks from the secondary profile. Do not combine profiles into a new creative lane.
