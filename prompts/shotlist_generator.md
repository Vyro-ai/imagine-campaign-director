# Shotlist Generator

Turn the brief into a production-ready shot list.

For each shot, include:

- shot number
- duration
- automation type
- purpose
- visual direction
- camera
- action
- text overlay
- audio/narration
- transition
- risk notes

Default 20-second structure:

1. Hook
2. Product/world reveal
3. Use-case or transformation
4. Detail/macro proof
5. Human/emotional moment
6. Offer/benefit
7. Product-first CTA
