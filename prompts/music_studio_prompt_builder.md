# Imagine.Art Music Studio Prompt Builder

Create music direction for Imagine.Art Music Studio.

## Required Fields

- campaign title
- music use: background score, full song, vocal hook, instrumental bed, trailer cue
- genre
- mood
- tempo / pace
- instrumentation
- vocal direction: instrumental, sung hook, or full song
- lyric direction if vocals are used
- duration target
- edit hit points
- sound-first status: visual-led score or sound-led spine
- beat ladder: major hits, silences, holds, reveals, and final absence/hit
- avoid list

## Sound-First Mode

Use this mode when the user supplies music, tempo, rhythm, voice, genre, or the creative gate selects a sound-led premise.

Before the shot ladder, define:

- sonic object or instrument that owns the campaign, such as bottle-on-stone, zipper teeth, breath, fluorescent buzz, heel scrape, shutter, sub drop, room tone, or vocal fragment
- tempo or pulse
- first audible hook
- beat ladder for cuts, holds, reveals, and silence
- final memory sound
- forbidden musical moves

The output must be strong enough that the visual shot ladder can be written against it. Do not translate visuals into music until after this path has been considered.

## Visual-To-Music Translation

If the user supplies only visual inputs:

- dark contrast -> lower register, restrained rhythm
- glossy chrome -> precise electronic or glassy synth texture
- warm skin highlights -> soft analog warmth
- streetwear flash -> punchy percussion, sparse bass
- fragrance nocturne -> slow atmospheric pulse
- industrial precision -> mechanical percussion, clean bass, tight transient design

## Output

Return one ready-to-paste Music Studio prompt plus optional lyrics if needed.
